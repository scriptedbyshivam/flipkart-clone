## 📖 About The Project

✨ This project is a **Flipkart Clone**, built to replicate the core UI and functionality of a real-world E-Commerce platform.  

💡 It demonstrates how modern shopping websites work — including product listings, categories, and responsive layouts.

🧠 The main goal of this project:
- Understand **Frontend Development**
- Learn **UI/UX Design**
- Practice **Real-world project building**

🔥 Key Highlights of this project:

- 🏠 Flipkart-style Homepage UI

- 🎯 Clean & Structured Design
